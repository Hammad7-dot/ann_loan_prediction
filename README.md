# Loan Approval Prediction Using ANN

## Objective
Build an Artificial Neural Network (ANN) model to predict loan approval status.

## Steps Performed
1. Dataset Loading
2. Missing Value Handling
3. Categorical Encoding
4. Feature Scaling
5. Train-Test Split
6. ANN Model Building
7. Model Training
8. Performance Evaluation

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-Score
